## Contributing Guidelines

1. When adding any new textures/models only add your own assets unless explicit permission has been granted.
   - If the latter is the case, provide proof of the permission within your Pull Request
2. Do not add any textures/models that were made through the use of generative A.I.
3. Always include `"author"` tags in any new models or item/block definitions. 
   - If you are modifying a pre-existing asset by a different author, add your name after there own, separated by commas.
